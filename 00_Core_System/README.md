# 00_Core_System - 核心系统

## 目录说明

存放 AI 助手的核心配置文档，定义系统的基础协议、记忆机制和个性设定。

## 文件说明

- `soul.md` - 个人设定：定义 AI 助手的特长、风格与关注内容
- `memory.md` - 记忆系统：长期记忆和上下文管理
- `ai_protocol.md` - AI 协议：交互规则和行为准则
- `skills_create_protocol.md` - 技能创建协议：自定义技能的开发规范
- `knowledge_map.md` - 知识地图：知识体系结构

## 重要性

这些文档是整个知识库的"操作系统"，定义了 AI 助手的核心行为模式。
