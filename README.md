# nightsailing 知识库

## 目录说明

这是一个个人知识管理系统，用于存放 AI 相关的核心系统文档、技能脚本、项目资料和学习笔记。

## 目录结构

- `00_Core_System/` - 核心系统文档（AI 协议、记忆系统、灵魂设定）
- `01_Active_Projects/` - 进行中的项目和调研
- `02_Prompts_Knowledge/` - 提示词知识库
- `03_Vision_Knowledge/` - 视觉相关知识和工具
- `04_Agent_Knowledge/` - AI Agent 相关知识
- `05_Model_Knowledge/` - 大语言模型知识
- `06_Inputs_and_Archive/` - 输入资料和归档内容
- `07_Think_and_Output/` - 思考过程和输出成果
- `08_skills/` - 自定义技能脚本

## 使用方式

这是一个 Git 仓库，可以在多设备间同步：

```bash
# 克隆到新设备
git clone https://github.com/xuelinf/nightsailing.git

# 拉取更新
git pull

# 提交更改
git add .
git commit -m "更新说明"
git push
```
