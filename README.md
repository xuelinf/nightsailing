# 夜航船之书

夜航船之书是临风的 AI 工具作品站，用来展示已经发布、正在打磨和计划开发的个人工具。

当前版本 `0.0.3` 包含：

- `AI航行日记`：macOS 菜单栏 token 用量工具，可下载，已开源。
- `问玄`：六爻铜钱起卦与 AI 解卦托盘工具，可下载。
- `端口占用管理器`：计划中的本机端口管理工具。
- `AI 语音输入`：计划中的免费开源 ASR 输入工具。

`0.0.3` 在 `0.0.2` 的作品站基础上，系统调研并转译了 Laper.ai 的展示方法：英雄区加入“夜航札记”输入器、漂浮产品页和证据型产品卡片；每个产品详情页也加入 prompt 场景、状态证据和更清晰的媒体停靠位。

调研记录保存在 `docs/research/laper-ai/laper-design-study.md`。公开仓库只保留研究结论，不保存第三方站点的原始截图或完整页面抓取。

## Typography

站点品牌标题与叙事标题使用开源字体 `LXGW WenKai Screen`（霞鹜文楷屏幕版），正文和界面控件使用中文现代黑体栈。

字体依据 SIL Open Font License 1.1 分发，许可证见 `OFL-LXGW-WenKai.txt`。
