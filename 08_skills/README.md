# 08_skills - 自定义技能

## 目录说明

存放自定义的 Claude Code 技能脚本，用于扩展 AI 助手的能力。

## 技能列表

### feishu-assistant
飞书助手：向团队成员/群组发消息、创建和更新文档、读取知识库文章、查看群消息、获取通讯录。

**主要功能：**
- 发送消息（文本/富文本/图片）
- 群聊管理（创建/添加成员/解散）
- 文档操作（创建/更新）
- 知识库阅读
- 通讯录管理
- 日历管理

### wechat-desktop
微信桌面自动化：基于 OCR 和桌面自动化技术实现微信消息读取和发送。

**主要功能：**
- 读取聊天内容
- 发送消息
- 后台监控
- 关键词触发
